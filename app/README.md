## `json-server` basic requests


|Method |Path       |Action             |
| ---   | ---       | ---               |
|GET	|/todos	    |get all todos      |
|GET	|/todos/id	|get Todo by id     |
|POST	|/todos	    |add new Todo       |
|PUT	|/todos/id	|update Todo by id  |
|DELETE	|/todos/id	|remove Todo by id  |
